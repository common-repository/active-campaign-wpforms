== Description ==
This plugin is a easy to use addon for WPform which helps to send form data to ActiveCampaign lists with very less configuration and easy to use interface.

Documentation: <a href="https://wpoperation.com/wp-documentation/active-campaign-wpforms/">View Online</a>

If you are looking to integrate Active Campaign with Contact Form 7 then here is plugin link: <a href="https://wordpress.org/plugins/wpop-accf/">Active Campaign & Contact Form 7</a>

=== Features === 

Adds subscription in "Active Campaign" through "WPform"
Option to select "Active Campaign" list ID for subscribers
Option to select "WPform" fields for "Active Campaign" list
Constant updates
Translation ready

== Installation ==

1. Unzip ac-wpforms.zip
2. Upload all the files to the /wp-content/plugins/ac-wpforms
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Install and activate WPforms plugin
